# CL Circular - Dashboard Pesquero

App web en espanol con 3 tabs:
- KPIs
- Empresas
- Serie de tiempo

## Ejecutar

1. Abre una terminal en esta carpeta.
2. Inicia un servidor estatico (ejemplo):
   - `npx serve .`
3. Abre la URL local que te muestre el comando.

## Archivos

- `index.html`: estructura de la app
- `styles.css`: estilo y colorimetria
- `app.js`: carga de datos, KPIs y graficas
- `CONAPESCA_BASE_UNIFICADA_2005_2024 (2).xlsx`: fuente principal de datos
- `empresas_pescados_mariscos_mexico.numbers`: referencia para tab Empresas
